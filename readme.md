# Fabspector - Textile Defect Detection

This project is a web-based application for detecting defects in textile fabrics. It utilizes a deep learning model to identify various types of defects from uploaded fabric images.

## 🚀 Features

*   **Image Upload**: Users can upload images of fabric for inspection.
*   **Defect Detection**: The system processes the uploaded images using a trained model to identify defects.
*   **Batch Processing**: Supports processing multiple images in a single batch.
*   **Report Generation**: Generates a PDF report detailing the detected defects.
*   **User-Friendly Interface**: Simple and intuitive web interface for easy operation.

## 🛠️ Technologies Used

*   **Backend**: Python, Flask
*   **Machine Learning**: TensorFlow, Keras
*   **Frontend**: HTML, CSS, JavaScript
*   **Deployment**: Designed for local deployment.

## ⚙️ Setup and Usage

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd Fabspector-HTML
    ```

2.  **Create and activate a virtual environment:**
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows use `venv\Scripts\activate`
    ```

3.  **Install the dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Run the application:**
    ```bash
    python app.py
    ```

5.  Open your web browser and navigate to `http://127.0.0.1:5000`.
